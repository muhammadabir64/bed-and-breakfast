# B&B - Hotel Management
A Flask-based Hotel Room Management Web App simplifies booking. Users can register/login, browse rooms, book stays, and securely pay with Stripe. It offers instant confirmations, user dashboards, and admin controls for a seamless booking experience.

#### Demo URL: https://bed-and-breakfast.onrender.com
#### Admin URL: https://bed-and-breakfast.onrender.com/admin_dashboard
